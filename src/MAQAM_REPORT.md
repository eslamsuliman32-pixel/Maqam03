# تقرير Maqam v2.0: هندسة الصوت والإيقاع النخبوية

## 1. محرك Maqam Phonetic (الصوتيات)
**الوظيفة:** المحرك الجوهري الذي يجمع بين علم العروض التقليدي وخوارزميات المعالجة الرقمية. يقوم بتحليل المقاطع الصوتية العربية لكل "بار" ويحولها إلى متجهات رقمية تعبر عن "الوزن الصوتي".
**المميزات:**
- تحليل أوزان "المورا" (Mora) بناءً على مخارج الحروف.
- التفريق بين الحروف الثقيلة (الانفجارية) والحروف اللينة (السائلة).
- حساب الكثافة اللفظية في الثانية الواحدة.

```typescript
// مقتطف من خوارزمية أوزان المقاطع الصوتية العربية
const ARABICMORAWEIGHTS: Record<string, number> = {
  ق: 1.8, ك: 1.7, ط: 1.8, ض: 1.9, // حروف ثقيلة
  ب: 1.2, د: 1.2, ف: 1.2,          // حروف متوسطة
  م: 0.9, ن: 0.9, ل: 0.9, و: 0.8  // حروف خفيفة
};

function calcMoraWeight(syllable: string): number {
  let total = 0;
  for (const char of syllable) {
    total += ARABICMORAWEIGHTS[char] || 1.0;
  }
  return total / syllable.length;
}
```

## 2. استوديو التحليل الموسيقي
**الوظيفة:** لوحة التحكم المركزية التي تدمج جميع الأدوات في بيئة عمل واحدة. تتيح للمستخدم تحميل الإيقاع (Beat)، تحليل النص، ومزامنة "البارات" مع الضربات الموسيقية بدقة ميلي-ثانية.

---

## 3. الوحدات الإيقاعية (Bars, Sections, BPM)
- **Bars (البارات):** الوحدة البنائية الأساسية للمسار. يتم هندسة كل بار بشكل مستقل لضمان تماسك التدفق (Flow).
- **Sections (الأقسام):** تقسيم الهيكل الهيكلي للأغنية (Intro, Verse, Hook, Outro).
- **BPM (السرعة):** المقياس الزمني الذي يتم بناء عليه الشبكة الإيقاعية.

---

## 4. المخطط الزمني للهيكل (Structure Timeline)
**الوظيفة:** عرض بانورامي للهيكل الزمني للأغنية. يسمح بسحب وإفلات الأقسام وتحديد المناطق الزمنية لكل قسم، مما يوفر رؤية سينمائية لتطور المسار الإبداعي.

---

## 5. الشبكة الإيقاعية (Rhythmic Grid)
**الوظيفة:** ميكروسكوب زمني يحلل كل ضربة (Beat) ويقسمها إلى خلايا فرعية. يتم توزيع المقاطع الصوتية على هذه الشبكة لضمان أن كل مخرج حرف يقع في موضعه الصحيح بالنسبة للإيقاع الراقص.

```typescript
// بناء خلايا الشبكة الإيقاعية
function buildGridCells(analysis: AudioAnalysisResult, syllables: string[]) {
  return analysis.beats.map((beat, i) => ({
    barIndex: Math.floor(i / 4),
    strength: beat.strength, // downbeat, strong, weak
    syllable: syllables[i] || "",
    moraWeight: calcMoraWeight(syllables[i])
  }));
}
```

## 6. خريطة الطاقة (Sonic Semantic Heatmap)
**الوظيفة:** نظام رادار عاطفي يحلل "الطيف الصوتي" للكلمات. يقيس مدى توافق الكلمات المختارة مع "الحالة الشعورية" المستهدفة (انفجاري، ناعم، عدواني، حزين).
**المعايير السبعة:** (ثقيل، متوسط، ناعم، احتكاكي، أنفي، سائل، حلقي).

## 7. استراتيجية السرد ومنحنى الطاقة الدرامية
**الوظيفة:** (Narrative Arc Panel) رسم بياني يوضح تطور القوة التعبيرية والتوتر الدرامي عبر المسار. يساعد الكاتب على تصميم "الانفجارات اللفظية" (Punchlines) في اللحظات الموسيقية الصحيحة.

## 8. هندسة القافية (Rhyme Engineering)
هذا هو القسم الأكثر دقة في النظام، حيث يتعامل مع "جيوب القافية" (Rhyme Slots) وتصنيفاتها الملكية:

### الأنواع الرئيسية للجيوب:
1.  **هبوط (Landing):** النقطة الرئيسية لسقوط القافية (على الضربة القوية الأولى).
2.  **نفَس (Breath):** موضع استراحة محدد مدروس بعناية بين الأقسام.
3.  **جيب (Pocket):** قافية داخلية تقع في "جيوب" الإيقاع غير المتوقعة (بين الضربات).
4.  **ظل (Ghost):** قافية خفية جداً تظهر على الضربات الضعيفة (Off-beats).
5.  **مرساة (Anchor):** قافية ثقيلة تعيد ضبط تدفق البار.
6.  **صدى (Echo):** تكرار صوتي خفيف يعزز التأثير النفسي.

### مجموعات القافية وتحليلها:
يتم تنظيم القوافي في مجموعات (A, B, C...) مع حساب "تطابقات صوتية" (Phonetic Matches) دقيقة.

```typescript
// منطق توزيع جيوب القافية
const SLOTCONFIG = {
  landing: { arabicLabel: "هبوط", color: "#F59E0B", intensity: 1.0 },
  breath:  { arabicLabel: "نفَس", color: "#06B6D4", intensity: 0.4 },
  pocket:  { arabicLabel: "جيب", color: "#10B981", intensity: 0.7 },
  ghost:   { arabicLabel: "ظل", color: "#8B5CF6", intensity: 0.3 }
};

// حساب نسبة التطابق
function checkRhymeMatch(a: string, b: string): boolean {
  if (!a || !b) return false;
  return a.slice(-2) === b.slice(-2); // مطابقة آخر صوتين
}
```

## 9. مستودع البارات (Bar Reservoir)
**الوظيفة:** قاعدة بيانات مركزية ذكية لحفظ جميع البارات المكتوبة، مع تصنيفها تلقائياً بناءً على "المشاعر"، "القافية"، و"الوزن الصوتي".
**المميزات:**
- الفلترة المتقدمة بناءً على "النبْر الإيقاعي".
- البحث عن "البارات المشابهة" باستخدام خوارزميات التعلم الآلي.
- التصدير المباشر لورشة الإنتاج.
